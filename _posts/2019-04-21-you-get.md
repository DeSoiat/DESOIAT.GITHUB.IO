# you-get 安装&使用

You-Get 是一个微型命令行实用程序，用于从 Web 下载媒体内容（视频、音频、图像）

# 安装


第一步 

从Python官网下载Python https://www.python.org/downloads/

---
第二步

安装Python 

这里一定 要勾选 Add Python to PATH 要不然CMD会报错

```bash
'python' 不是内部或外部命令，也不是可运行的程序
或批处理文件。
```
